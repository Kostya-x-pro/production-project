* npm init -y  - команда разворачивает package.json (с подтвержденными дефолтными настройками)
* npm i ..... - установить webpak в проект и webpakcli

* webpack.config.ts - главный файл настройки вебпака
* webpack - команда в терменале запускающая webpack

* npm run start - Комадна для запуска режима работы с проектом
* npm run build:prod - Комадна для запуска билдва в продакшен версии
* npm run build:dev - Комадна для запуска запуска билда в режиме разработки

* webpackDevServer - нужен для того что бы автоматически перезапускать сборку проекта

module.exports - старый синтаксис сборки comonJS
